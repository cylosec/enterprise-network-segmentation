# Project Overview

## Scenario

This project simulates a network discovery and assessment engagement for a university engineering 
department.

The objective is to document network resources, identify security and operational considerations, 
and evaluate segmentation strategies using VLANs, subnetting, inter-VLAN routing, and access 
control policies.

The assessment focuses on understanding how faculty, administration, student labs, research 
systems, servers, guest resources, and IT management systems communicate across the network and 
how segmentation can support future security initiatives.

## Technologies

* Cisco Packet Tracer
* VLANs
* Subnetting
* 802.1Q Trunking
* Router-on-a-Stick
* Access Control Lists (Future Phase)
* DHCP Services (Future Phase)

## Project Phases

1. Discovery & Assessment
2. VLAN Planning
3. Subnetting
4. Inter-VLAN Routing
5. Connectivity Validation
6. Security Recommendations
